Introdução

Este é um blog de formação do curso da HTML da DIO.me, onde você encontrará seções com o conteúdo das aulas feitas no curso, sobre Estruturação, semântica e as principais tags do HTML. Durante o percurso, veremos como criar textos com títulos, parágrafos, destaques; como criar listas, links, imagens, tabelas, formulários, além de aprender um pouco sobre formatação via CSS, tags estrurais, semântica e até acessibilidade.

Para ir diretamente ao assunto desejado, basta clicar nos liks presentes no índice acima. Para retornar ao início, basta clicar no botão "voltar" que ficará abaixo do título do respectivo capítulo.

Antes de falarmos sobre as tegs, é importante entender algumas noções básicas de HTML: ele é um linguagem de marcação que é usada para estruturar o conteúdo de uma página Web. Atualmente, usamos o HTML 5. Em todas suas versões, o HTML segue padrões que são definidos pela W3C. Mas, por hora, é importante saber que todas as tegs HTML são escritas entre sinais de menor e maior (< e >) e a maioria delas possui uma teg de abertura e uma de fechamento (que é a mesma coisa da de abertura, mas com uma barra antes do nome da teg).
